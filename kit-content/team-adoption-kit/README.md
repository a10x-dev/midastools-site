## How to Use This Kit

### Step 1 — Audit (Week 0)
Run `10-onboarding-enablement.md → Skill Discovery Guide` with each department head. You'll surface the 3–5 workflows per team that are best candidates for AI assist.

### Step 2 — Baseline (Week 1)
Run `09-adoption-framework-templates.md → AI Proficiency Assessment` across every employee. Score each person L0–L3. This is your before-number for the ROI report you'll show the CEO in 90 days.

### Step 3 — Activate (Week 2–4)
Send each department their playbook file (files 01–08). Each playbook is a self-contained "here are the 6 things your team should be doing with AI this quarter." Pair with the Day 1 Setup Guide for each new user.

### Step 4 — Track (Ongoing)
Stand up the Weekly Usage Leaderboard and AI Skills Marketplace from file 09. Make adoption visible, not enforced.

### Step 5 — Report (Day 90)
Run the ROI Calculator + Quarterly Adoption Report from file 09. This is what you send to the exec team to justify Phase 2 budget.

## How Variables Work

Every prompt uses `[BRACKET NOTATION]` for customizable fields:

```
[COMPANY NAME] — Your company name
[DEPARTMENT] — Engineering, Sales, Marketing, etc.
[TEAM SIZE] — Number of people in the team
[TOOL] — Which AI tool (Claude, ChatGPT, Gemini, Copilot)
[ROLE] — Specific job title being addressed
```

Replace these before pasting into your AI tool. Specificity in inputs = quality in outputs.

## Works With

- Claude (Sonnet 4.5, Opus 4.7) — best for analysis, doc review, long-context synthesis
- ChatGPT (GPT-5, GPT-4o) — best for sales scripts, creative copy, brainstorming
- Google Gemini (2.5 Pro) — best for data analysis + spreadsheet workflows
- Microsoft Copilot — for teams already in M365
- Any LLM that accepts text prompts

If your company has a primary tool, standardize on it for the rollout. Tool sprawl kills adoption.
