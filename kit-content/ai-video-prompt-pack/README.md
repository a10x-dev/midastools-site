## Pro Tips

- **Duration**: Specify length (e.g., "5-second clip") for better control
- **Camera movement**: Include specific camera instructions (pan, zoom, dolly, tracking shot)
- **Consistency**: Use the same style keywords across clips for cohesive projects
- **Aspect ratio**: Specify 9:16 for vertical (TikTok/Reels) or 16:9 for horizontal (YouTube)
- **Seed values**: Save seed numbers to reproduce similar styles
