## Prompt 1: Sales Email — Cold Outreach

### Prompt

```
Write a cold outreach email for [PRODUCT/SERVICE] targeting [TARGET AUDIENCE].

Context:
- The main pain point I solve: [PAIN POINT]
- My key differentiator: [BENEFIT]
- Price point: [PRICE]

Rules:
- Subject line must create curiosity without being clickbait
- Opening line must reference something specific to their role (not "I hope this finds you well")
- Body must be under 125 words
- Include exactly one specific metric or result
- CTA must be a soft ask (question, not demand)
- No attachments, no links in first email
- Write 3 variations: direct, story-based, and data-led
```

### How to Use
Send variation A to 33% of your list, B to 33%, C to 33%. Track open rates and reply rates. Double down on the winner.

## Prompt 2: Sales Email — Follow-Up Sequence (5 Emails)

### Prompt

```
Write a 5-email follow-up sequence for prospects who didn't reply to my cold email about [PRODUCT/SERVICE].

Original email context:
- Product: [PRODUCT/SERVICE]
- Audience: [TARGET AUDIENCE]
- Key benefit: [BENEFIT]

Sequence timing and angles:
- Email 2 (Day 3): Share a relevant insight or stat — no pitch
- Email 3 (Day 7): Brief case study or result — social proof
- Email 4 (Day 14): Different angle on the same pain point
- Email 5 (Day 21): Breakup email — "closing your file" with one final soft CTA

Rules for ALL emails:
- Under 80 words each
- Each email must stand alone (don't reference previous emails)
- No guilt-tripping or passive aggression
- Each must provide independent value even if they never buy
- Subject lines should NOT contain "following up" or "checking in"
```

## Prompt 3: Landing Page Copy — Above the Fold

### Prompt

```
Write above-the-fold landing page copy for [PRODUCT/SERVICE].

Product details:
- What it does: [PRODUCT/SERVICE DESCRIPTION]
- Who it's for: [TARGET AUDIENCE]
- Main transformation: [BENEFIT]
- Price: [PRICE]
- Social proof: [TESTIMONIAL OR METRIC]

Write these elements:
1. Headline (under 10 words, outcome-focused, no jargon)
2. Subheadline (one sentence expanding on the headline)
3. 3 bullet points (benefit-driven, each starting with a verb)
4. CTA button text (action-oriented, 4 words max)
5. Trust line below CTA (guarantee, security, or social proof)

Write 3 variations:
- Version A: Pain-focused (lead with the problem)
- Version B: Outcome-focused (lead with the transformation)
- Version C: Social-proof-focused (lead with results)
```

## Prompt 4: Landing Page Copy — Full Page

### Prompt

```
Write a complete long-form landing page for [PRODUCT/SERVICE] targeting [TARGET AUDIENCE].

Include these sections in order:
1. Hero: Headline + subheadline + CTA
2. Problem: Describe the pain of NOT having this solution (3 specific scenarios)
3. Solution: Introduce the product as the answer (features → benefits)
4. Social Proof: Format [TESTIMONIAL] as a compelling case study block
5. What's Included: Bullet list of everything they get
6. Objection Handling: Address top 3 objections with rebuttals
7. Pricing: Present [PRICE] with value anchoring (compare to alternatives or DIY cost)
8. FAQ: 5 questions a skeptical buyer would ask
9. Final CTA: Urgency + guarantee + button

Tone: Confident but not pushy. Conversational but professional.
Each section should be scannable — use short paragraphs, bold key phrases, and bullet points.
```

## Prompt 7: Sales Script — Discovery Call

### Prompt

```
Write a discovery call script for selling [PRODUCT/SERVICE] to [TARGET AUDIENCE].

Product context:
- What I sell: [PRODUCT/SERVICE]
- Price: [PRICE]
- Common objections: [LIST 2-3 OBJECTIONS]
- Ideal outcome: [WHAT SUCCESS LOOKS LIKE FOR BUYER]

Structure the script as:
1. Opening (30 sec): Build rapport, set agenda, get permission to ask questions
2. Discovery (5 min): 5 open-ended questions to uncover pain, urgency, budget, timeline, decision process
3. Diagnosis (2 min): Summarize their situation back to them (show you listened)
4. Prescription (3 min): Present solution mapped to THEIR specific pain points
5. Objection Handling: Write rebuttals for each common objection
6. Close (1 min): Soft close with clear next step

Include exact phrases. Mark where to PAUSE and LISTEN (most people talk too much on sales calls).
```

## Prompt 8: Case Study Writer

Context:

- Most common reason people don't buy: [OBJECTION]

## Prompt 12: Sales Objection Rebuttals

### Prompt

```
I sell [PRODUCT/SERVICE] at [PRICE] to [TARGET AUDIENCE].

These are the objections I hear most:
1. [OBJECTION 1 — e.g., "It's too expensive"]
2. [OBJECTION 2 — e.g., "I need to think about it"]
3. [OBJECTION 3 — e.g., "I'm not sure it'll work for my situation"]
4. [OBJECTION 4 — e.g., "I can do this myself"]
5. [OBJECTION 5 — e.g., "I need to talk to my partner/team"]

For each objection, write:
- The REAL concern behind it (what they're actually worried about)
- A 2-sentence acknowledgment (validate, don't dismiss)
- A rebuttal using the Feel-Felt-Found framework
- A specific example, stat, or story that neutralizes it
- A bridge question that moves the conversation forward

Tone: Empathetic and confident. Never argue or get defensive.
```

Details:

   - Customer jobs (what they're trying to do)

## Prompt 18: Guarantee Copy

### Prompt

```
Write guarantee copy for [PRODUCT/SERVICE] at [PRICE].

Guarantee type: [30-day money-back / satisfaction / results-based]

Write 3 versions:
1. Short (for product pages): 2-3 sentences, confidence-building
2. Medium (for checkout pages): 1 paragraph explaining the guarantee terms clearly
3. Bold (for sales pages): Full guarantee section with headline, body, and visual description

Rules:
- Name the guarantee (e.g., "The No-BS 30-Day Guarantee")
- Be specific about what happens if they're not satisfied
- Remove ALL risk from the buyer
- Tone should be confident, not defensive
- Include a line about WHY you offer this guarantee (shows confidence in product)
```

## Pro Tips

- **Test everything**: Write 3 headline variations, send to small segments, scale the winner
- **Specificity sells**: "$14,718 in 3 weeks" beats "thousands of dollars"
- **One CTA per piece**: Every email, ad, and page should have ONE clear next step
- **Read it out loud**: If it sounds like a robot wrote it, add conversational phrases
- **Swipe file**: Save the best outputs. Build a library of proven copy you can adapt
