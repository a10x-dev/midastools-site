# 03: Content Creation Prompts

Create blogs, newsletters, video scripts, podcasts, and lead magnets in a fraction of the time. These prompts help you produce high-quality content that builds authority and drives traffic.

## Prompt 1: Blog Post — SEO Optimized

### Prompt

```
Write a comprehensive blog post targeting the keyword "[KEYWORD]" for [TARGET AUDIENCE].

Context:
- Industry: [INDUSTRY]
- Content goal: Rank on page 1 for "[KEYWORD]" and drive traffic to [PRODUCT/SERVICE]
- Word count: [1,500-2,500 WORDS]
- Reader expertise: [EXPERTISE LEVEL]

Requirements:
1. Title: Include exact keyword, under 60 characters, compelling enough to click
2. Meta description: 155 characters, includes keyword, has a CTA
3. H2/H3 structure: At least 5 H2s and 3 H3s (Google loves structured content)
4. Introduction (100 words): Hook → problem → promise → overview
5. Body sections: Each H2 should be a complete mini-article (300-400 words)
6. Include: at least 3 specific examples, 2 statistics, 1 expert quote
7. Conclusion: Summarize key points → CTA to [PRODUCT/SERVICE]
8. Internal link suggestions: Where to link to related content

SEO requirements:
- Use "[KEYWORD]" in the first 100 words
- Include 3-5 semantically related keywords naturally
- Use question-based H2s where possible (targets featured snippets)
- Include a "Key Takeaways" or "TL;DR" section (targets featured snippets)
- Alt text suggestions for 3 images
```

## Prompt 2: Blog Post — Listicle (High-Traffic Format)

### Prompt

```
Write a listicle blog post: "[NUMBER] [THINGS] for [TARGET AUDIENCE] in [YEAR]"

Topic: [TOPIC]
Target keyword: [KEYWORD]
Number of items: [7-15 IS OPTIMAL]
Audience: [TARGET AUDIENCE]

For each item:
1. Bold headline with item name
2. 2-3 sentence description of what it is
3. Why it matters (specific benefit or stat)
4. How to implement/use it (actionable step)
5. Pro tip or insider insight

Structure:
- Introduction: Why this list matters NOW (urgency/relevance)
- Items: Ordered from most impactful to least (readers often don't finish)
- Conclusion: Top 3 picks for different scenarios + CTA
- "What did we miss?" line at the end (drives comments)

Word count: [1,500-2,000 WORDS]
Include a summary table at the top (targets featured snippets).
```

## Prompt 3: Newsletter — Weekly Edition

### Prompt

```
Write a weekly newsletter edition for [YOUR NAME/BRAND] about [TOPIC].

Newsletter context:
- Audience: [TARGET AUDIENCE]
- Voice: [BRAND VOICE]
- This week's main insight: [MAIN TOPIC OR STORY]
- 3 links/resources to share: [LINK 1, LINK 2, LINK 3]
- Product to promote: [PRODUCT/SERVICE]

Structure:
1. **Subject line** (3 variations): Must get opened. Use curiosity, benefit, or urgency.
2. **Preview text**: 40-90 characters, complements the subject line
3. **Personal opener** (2-3 sentences): Something real from your week — builds connection
4. **Main insight** (200-300 words): The one big idea worth their time
5. **3 Quick Links**: Each with a 1-sentence takeaway (not just a URL dump)
6. **One actionable tip**: Something they can implement in under 5 minutes
7. **Soft CTA**: Natural transition to [PRODUCT/SERVICE]
8. **Sign-off**: Warm, personal, not corporate

Total length: Under 500 words (respect their inbox).
Mobile-optimized: Short paragraphs, no wide images.
```

## Prompt 4: YouTube Video Script

### Prompt

```
Write a YouTube video script about [TOPIC] for [TARGET AUDIENCE].

Channel context:
- Niche: [INDUSTRY]
- Video style: [talking head / tutorial / animated / documentary]
- Target length: [8-12 MINUTES — optimal for ad revenue and retention]
- Main keyword: [KEYWORD]

Script structure:
1. **Hook (0:00-0:30)**: The first 30 seconds determine everything. Open with a bold claim, surprising stat, or "In this video, you'll learn [specific outcome]." NO intros, NO "Hey guys!", NO channel bumpers.
2. **Setup (0:30-1:30)**: Why this matters. What's at stake. Create buy-in.
3. **Content sections (1:30-9:00)**: 3-5 main points, each with:
   - Transition line
   - Key insight
   - Example or story
   - Visual/B-roll suggestion
4. **Recap (9:00-10:00)**: Summarize the 3 main takeaways
5. **CTA (10:00-10:30)**: Subscribe + related video + resource in description

Include:
- Retention hooks every 2 minutes ("But here's where it gets interesting...")
- Timestamp suggestions for chapter markers
- Description text (200 words, keyword-rich, with links)
- 5 title variations (ranked by click potential)
- Thumbnail concept (text overlay + visual)
- 10 tags for SEO
```

## Prompt 5: Podcast Episode Outline

### Prompt

```
Create a detailed podcast episode outline about [TOPIC].

Podcast context:
- Show name: [PODCAST NAME]
- Format: [solo / interview / co-hosted]
- Typical length: [DURATION]
- Audience: [TARGET AUDIENCE]
- This episode's goal: [EDUCATE / ENTERTAIN / PROMOTE / INTERVIEW]

Outline:
1. **Cold open** (30 sec): Start with the most interesting insight from this episode (hooks listeners immediately)
2. **Intro** (60 sec): Brief show intro + what this episode covers + why it matters now
3. **Segment 1** (10-15 min): [SUBTOPIC] — key points, stories, examples
4. **Segment 2** (10-15 min): [SUBTOPIC] — key points, stories, examples
5. **Segment 3** (5-10 min): [SUBTOPIC] — key points, stories, examples
6. **Key takeaways** (2 min): 3 bullets the listener can act on today
7. **CTA** (30 sec): Subscribe, review, share, or visit [LINK]

Include:
- Talking points for each segment (not a full script — podcasts should sound natural)
- 2-3 anecdotes or stories to tell
- Listener engagement question for social media
- Episode title (3 variations)
- Episode description (100 words)
- Show notes with timestamps
```

## Prompt 6: Lead Magnet — Checklist/Guide

### Prompt

```
Create a lead magnet [checklist/guide/cheat sheet] about [TOPIC] for [TARGET AUDIENCE].

Context:
- What they get in exchange for their email: This [checklist/guide/cheat sheet]
- Related product: [PRODUCT/SERVICE] at [PRICE]
- The lead magnet should make them want the full product

Format:
1. **Title**: Clear, benefit-driven (e.g., "The [Number]-Step [Topic] Checklist")
2. **Subtitle**: Who it's for and what they'll achieve
3. **Introduction** (50 words): Why this matters + how to use it
4. **The content** (format depends on type):
   - Checklist: 15-25 items, grouped into 3-5 categories, with checkbox formatting
   - Guide: 5-7 steps with 2-3 sentences of explanation each
   - Cheat sheet: Quick-reference table or list, scannable in under 2 minutes
5. **Bonus tip**: One advanced insight that surprises them
6. **CTA**: "Want the complete system? Get [PRODUCT] →"
7. **About section**: 2 sentences about your brand + link

Design notes:
- Must fit on 1-3 pages (PDF)
- Must be immediately useful (they should get value in 60 seconds)
- Must create desire for the full product (taste, don't satisfy completely)
```

## Prompt 7: Email Course — 5-Day Drip

### Prompt

```
Write a 5-day email course about [TOPIC] for [TARGET AUDIENCE].

Context:
- Course title: [NAME]
- What they signed up for: [THE PROMISE — e.g., "5 days to better proposals"]
- What you're selling at the end: [PRODUCT/SERVICE] at [PRICE]

Day-by-day:
- **Day 1**: The Problem — Why [TOPIC] matters now + one mindset shift. End with: "Tomorrow, I'll show you the first step."
- **Day 2**: Quick Win — The easiest, fastest improvement they can make today. Build confidence.
- **Day 3**: The Framework — Your core methodology or approach. This is where you demonstrate expertise.
- **Day 4**: Common Mistakes — 3 things that hold people back (creates "I need help" feeling)
- **Day 5**: The Full System — How it all comes together + CTA to [PRODUCT/SERVICE]

For each email:
- Subject line (2 variations)
- Preview text
- Body (200-300 words)
- One specific action item

Rules:
- Each email must provide standalone value (even if they never buy)
- Build anticipation for the next day
- Day 5 transition to paid should feel natural, not jarring
- Total send time: over 5 days, same time each day
```

## Prompt 8: Content Repurposing Engine

### Prompt

```
I have this piece of content. Turn it into 8 different formats:

Original content:
[PASTE YOUR BLOG POST, TRANSCRIPT, OR ARTICLE]

Create:
1. **Tweet storm** (8 tweets): Key insights as a thread
2. **LinkedIn post**: Professional angle, story format
3. **Instagram carousel** (8 slides): One insight per slide, visual-friendly
4. **YouTube short script** (60 sec): Hook + key point + CTA
5. **Newsletter blurb** (100 words): Teaser for email list
6. **Podcast talking points** (5 bullets): For speaking about this topic
7. **Quora/Reddit answer**: Answer a question this content addresses
8. **Quote graphics** (5): Pull the most quotable lines for social images

Rules:
- Each format must feel NATIVE to its platform
- Don't just shorten — REIMAGINE for each medium
- Maintain the core insight but change the delivery
- Include platform-specific formatting (hashtags, emojis, line breaks)
```

## Prompt 9: Content Gap Analysis

### Prompt

```
Analyze my content strategy and identify gaps.

My current content:
- Topics I've covered: [LIST YOUR RECENT CONTENT TOPICS]
- Platforms I'm active on: [PLATFORMS]
- Content that performed best: [YOUR TOP 3 PIECES AND WHY THEY WORKED]
- Content that flopped: [YOUR BOTTOM 3 PIECES]
- My product: [PRODUCT/SERVICE]
- My audience: [TARGET AUDIENCE]

Provide:
1. **Content pillars audit**: Am I covering the right topics for my audience?
2. **Format gaps**: Formats I should try (e.g., if I only do blogs, maybe video?)
3. **Funnel gaps**: Am I creating content for every stage? (awareness → consideration → decision)
4. **Topic opportunities**: 10 topics my audience cares about that I haven't covered
5. **Keyword opportunities**: 10 keywords I should target based on my niche
6. **Competitive gaps**: Content my competitors likely produce that I don't
7. **90-day content plan**: Week-by-week themes and formats

Be specific. Don't just say "do more video" — tell me exactly what kind of video and why.
```

## Prompt 10: Article Outline — Deep Dive

### Prompt

```
Create a detailed article outline for a deep-dive piece on [TOPIC].

Context:
- Target keyword: [KEYWORD]
- Word count: [2,500-4,000]
- Audience: [TARGET AUDIENCE] at [EXPERTISE LEVEL]
- Angle: [WHAT MAKES YOUR TAKE DIFFERENT]

Outline:
1. **Title** (3 variations with keyword)
2. **TL;DR** (2 sentences — what they'll learn and why it matters)
3. **Introduction**: Hook → context → thesis → roadmap
4. **Section breakdown**: For each H2:
   - H2 heading (question format when possible)
   - Key argument/point (1 sentence)
   - Evidence needed (stat, example, expert quote)
   - H3 sub-sections if needed
   - Transition to next section
5. **Expert quote suggestions**: Who to cite and what kind of quote
6. **Visual suggestions**: Charts, diagrams, or images for each section
7. **Conclusion**: Summary → implications → CTA
8. **Meta data**: Title tag, meta description, slug, categories, tags

This should be thorough enough that a writer could draft the full article from this outline alone.
```

## Prompt 11: Headline A/B Testing Generator

### Prompt

```
I'm writing about [TOPIC] for [TARGET AUDIENCE].

Generate 20 headline variations I can A/B test:

Category breakdown:
- 4 How-to headlines
- 4 Number/listicle headlines
- 4 Question headlines
- 4 Bold claim headlines
- 4 Data-driven headlines

For each headline:
- Rate 1-5 on: Clarity, Curiosity, Specificity, SEO potential
- Flag whether it's better for [BLOG / EMAIL / SOCIAL / AD]
- Provide a one-line rationale for why it could work

Then pick your top 5 across all categories and explain why they'd outperform the rest.
```

## Prompt 12: Content Brief for Outsourcing

### Prompt

```
Write a content brief I can hand to a freelance writer for an article about [TOPIC].

Article details:
- Target keyword: [KEYWORD]
- Word count: [TARGET]
- Audience: [TARGET AUDIENCE]
- Goal: [TRAFFIC / LEADS / AUTHORITY / LINKS]
- Tone: [BRAND VOICE]
- Deadline: [DATE]

The brief should include:
1. **Working title** and 2 alternatives
2. **Target keyword** + 5 secondary keywords to include naturally
3. **Search intent**: What the reader is trying to accomplish
4. **Competitive analysis**: What the top 3 results cover (and what they miss)
5. **Required sections** with H2/H3 outline
6. **Must-include elements**: Stats, examples, expert quotes, internal links
7. **Tone and style guidelines**: Specific dos and don'ts
8. **CTA**: What the reader should do after reading
9. **SEO checklist**: Meta title, description, image alt text, internal links
10. **Examples of good content**: 2-3 links to similar content you admire

This brief should eliminate the need for back-and-forth revisions.
```

## Pro Tips

- **Front-load value**: Put your best insight in the first 20% of any content. Most people don't finish.
- **Write the headline first**: If you can't write a compelling headline, the content idea isn't strong enough.
- **One piece = one idea**: Don't try to cover everything. Deep beats wide.
- **Content compounds**: A blog post that takes 3 hours to write can drive traffic for 3 years. Think in terms of ROI per hour, not per piece.
- **Repurpose before creating**: Always ask "can I repurpose an existing piece?" before writing from scratch.
