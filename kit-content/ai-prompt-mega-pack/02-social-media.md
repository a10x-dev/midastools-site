# 02: Social Media Prompts

Stop staring at a blank screen. These prompts generate scroll-stopping posts, viral threads, engagement hooks, and content calendars for every major platform. Batch a week's content in 30 minutes.

## Prompt 1: Twitter/X Thread (Viral Format)

### Prompt

```
Write a 10-tweet Twitter thread about [TOPIC] targeting [TARGET AUDIENCE].

Context:
- My expertise/angle: [YOUR UNIQUE PERSPECTIVE]
- Goal: [CONTENT GOAL — e.g., establish authority, drive traffic, generate leads]

Thread structure:
- Tweet 1 (hook): Bold claim, surprising stat, or contrarian take. Must stop the scroll. End with "🧵" or "A thread:"
- Tweets 2-4: Build the argument with specific examples, data, or stories
- Tweets 5-8: Actionable steps, frameworks, or insights they can use immediately
- Tweet 9: Summarize the key takeaway in one sentence
- Tweet 10: CTA — follow, retweet, reply, or link

Rules:
- Each tweet under 270 characters (leave room for engagement)
- Use line breaks for readability
- No hashtags in the thread (they kill engagement on X)
- Include at least 2 specific numbers or examples
- Tweet 1 must work as a standalone post (it's what gets retweeted)
- Write in [BRAND VOICE] tone
```

## Prompt 2: LinkedIn Post — Thought Leadership

### Prompt

```
Write a LinkedIn post about [TOPIC] that positions me as an expert in [INDUSTRY].

Context:
- My role: [YOUR TITLE/ROLE]
- My audience: [TARGET AUDIENCE ON LINKEDIN]
- A recent experience or insight: [BRIEF DESCRIPTION]

Format (LinkedIn algorithm-optimized):
- Hook (first 2 lines — must compel the "see more" click): Start with a bold statement, question, or personal story
- Body (8-12 short lines): Share the insight, use "I" sparingly, make it about them
- Each line should be 1 sentence max (LinkedIn rewards white space)
- Include a pattern interrupt halfway through (a surprising twist or "But here's what nobody tells you:")
- End with a question that invites comments (LinkedIn boosts posts with comments in the first hour)

Rules:
- No emojis in the first line (looks spammy)
- Max 3 emojis total if used
- No hashtags in the body (put 3-5 at the very end if you must)
- No external links in the post (kills reach — put link in comments)
- Under 1,300 characters for mobile optimization
- Write 2 variations: story-led and insight-led
```

## Prompt 3: LinkedIn Post — Personal Story

### Prompt

```
Write a LinkedIn post sharing a personal professional story that my audience of [TARGET AUDIENCE] will relate to.

The story: [BRIEFLY DESCRIBE WHAT HAPPENED — a failure, a lesson, a turning point, a win]
The lesson: [WHAT YOU LEARNED]
How it connects to: [YOUR PRODUCT/SERVICE/EXPERTISE]

Structure:
- Line 1: Start mid-story (not "Last week I..." — start with dialogue or action)
- Lines 2-5: Set the scene quickly (who, what, stakes)
- Lines 6-8: The turning point or revelation
- Lines 9-11: The lesson extracted (make it universal, not just about you)
- Line 12: Ask a reflective question

Rules:
- Must feel authentic, not manufactured
- Show vulnerability without being self-pitying
- The lesson should be applicable to the reader's life
- Don't pitch — the connection to your work should be implicit
- Under 1,500 characters
```

## Prompt 4: Instagram Carousel — Educational

### Prompt

```
Write an Instagram carousel (10 slides) about [TOPIC] for [TARGET AUDIENCE].

Context:
- My niche: [INDUSTRY]
- Content goal: [SAVES and SHARES — these drive reach on Instagram]
- My brand voice: [BRAND VOICE]

Slide structure:
- Slide 1 (Cover): Bold headline that promises value. Use "How to..." or "[Number] ways to..." or a provocative question. No more than 8 words.
- Slides 2-9: One key point per slide. Headline (bold, 5-7 words) + supporting text (2-3 sentences max).
- Slide 10 (CTA): Summary + follow CTA + save reminder

Rules:
- Each slide must stand alone (people screenshot individual slides)
- Write for scanning — bold headers, short sentences
- Include at least 3 "save-worthy" tactical tips
- No fluff or generic advice — every slide must make them think "I need to remember this"
- Suggest an aesthetic direction: [minimalist, bold/dark, pastel, branded]

Also write:
- Caption (under 150 words): Hook + summary + CTA + 5 relevant hashtags
- Alt text for accessibility
```

## Prompt 5: Instagram Reels / TikTok Script

### Prompt

```
Write a 30-60 second Reels/TikTok script about [TOPIC] targeting [TARGET AUDIENCE].

Context:
- My niche: [INDUSTRY]
- My on-camera style: [talking head / voiceover / text-on-screen / skit]
- Goal: [AWARENESS / ENGAGEMENT / LEADS]

Script format:
- HOOK (first 3 seconds): The line that stops the scroll. Must create curiosity or shock. Start with "POV:", "Stop scrolling if...", a question, or a bold claim.
- SETUP (5-10 seconds): Context or problem statement
- VALUE (15-30 seconds): The actual insight, hack, or story
- PUNCHLINE/CTA (5-10 seconds): Unexpected twist, summary, or call to action

Include:
- On-screen text suggestions for each section
- Music/sound suggestion (trending or mood-based)
- Caption with 3-5 hashtags
- A hook variation in case the first doesn't land

Rules:
- Speak in second person ("you") not third person
- No long intros or "Hey guys!"
- Front-load the value — assume they'll swipe at second 4
- Every line must earn the next second of attention
```

## Prompt 6: Social Media Content Calendar (1 Week)

### Prompt

```
Create a 7-day social media content calendar for [YOUR NAME/BRAND] on [PLATFORM(S)].

Context:
- Industry: [INDUSTRY]
- Target audience: [TARGET AUDIENCE]
- Content pillars: [LIST 3-4 THEMES — e.g., education, behind-the-scenes, customer stories, industry news]
- Posting frequency: [X POSTS PER DAY PER PLATFORM]
- Product/service to promote: [PRODUCT/SERVICE]
- Upcoming events or launches: [ANY TIMELY HOOKS]

For each day, provide:
1. Post type (educational, storytelling, promotional, engagement, curated)
2. Platform(s)
3. Full caption/copy (ready to post)
4. Visual suggestion (what the image/graphic should show)
5. Hashtags (platform-appropriate)
6. Best time to post
7. Engagement prompt (question or CTA)

Mix: 70% value, 20% personal/story, 10% promotional (the 70-20-10 rule).
No more than 2 promotional posts in the entire week.
```

## Prompt 7: Social Proof Posts (5 Formats)

### Prompt

```
Turn this customer result into 5 different social media posts:

Customer result: [DESCRIBE THE RESULT — e.g., "Client went from 0 to $10K/month in 90 days"]
Product/service used: [PRODUCT/SERVICE]
Platform: [PLATFORM]

Create:
1. **The Metric Post**: Lead with the number. "From [X] to [Y] in [time]." Short, punchy, screenshot-worthy.
2. **The Story Post**: Tell the before/during/after narrative (under 200 words)
3. **The Quote Post**: Format the customer's words as a testimonial graphic caption
4. **The Lesson Post**: Extract 3 lessons from their success that anyone can apply
5. **The Comparison Post**: Before (without your product) vs. After (with it) — specific contrasts

For each, include the full caption, suggested visual, and 1 engagement hook.
```

## Prompt 8: Engagement Hooks — 20 Posts

### Prompt

```
Write 20 engagement-driving social media posts for [PLATFORM] in the [INDUSTRY] niche.

My audience: [TARGET AUDIENCE]
My voice: [BRAND VOICE]

Create 4 posts for each engagement type:
1. **Questions**: Open-ended questions that invite debate or personal sharing
2. **Polls/This-or-That**: Binary choices that people can't resist answering
3. **Fill-in-the-blank**: "The best [tool/strategy/decision] I ever [action] was ________"
4. **Controversial takes**: Mildly provocative opinions that split the audience (not offensive, just debatable)
5. **"Wrong answers only"**: Fun prompts that get creative responses

Rules:
- Each post should be under 100 words
- Must be relevant to [INDUSTRY] (not generic "What's your favorite color?")
- Should attract your TARGET audience, not random engagement
- Include the engagement CTA naturally (don't just say "comment below!")
```

## Prompt 9: Hashtag Research & Strategy

### Prompt

```
Create a hashtag strategy for [YOUR NAME/BRAND] on [PLATFORM] in the [INDUSTRY] niche.

Provide:
1. **Core hashtags (5)**: Directly describe what you do — use consistently
2. **Niche hashtags (10)**: Specific to your sub-niche — lower competition, higher relevance
3. **Community hashtags (5)**: Tags your target audience follows and engages with
4. **Trending/timely hashtags (5)**: Currently active hashtags you can ride
5. **Branded hashtag (1)**: A unique tag for your content/community

For each hashtag:
- Estimated post volume (small <10K, medium 10K-500K, large 500K+)
- Relevance score (1-5)
- Recommended use frequency

Also provide:
- Platform-specific best practices (how many hashtags, where to place them)
- 3 hashtag sets I can rotate (to avoid repetition penalties)
```

## Prompt 10: Comment & DM Response Templates

### Prompt

```
Write response templates for common social media interactions in my [INDUSTRY] business.

My product/service: [PRODUCT/SERVICE]
My brand voice: [BRAND VOICE]

Create templates for:
1. **Positive comment**: Someone praises your content → engage genuinely + invite them to DM
2. **Question about pricing**: Someone asks "How much?" → respond without being pushy
3. **Skeptic/hater**: Someone doubts your claims → professional, data-backed response
4. **"How do I start?"**: Someone wants advice → give one tip + point to your resource
5. **Collaboration request**: Someone wants to partner → qualify before committing
6. **"Do you offer X?"**: Service/product inquiry → answer + soft upsell
7. **Thank you after purchase**: Customer shares their win → amplify + ask for review
8. **Cold DM response**: Someone reaches out cold → warm but boundaried response
9. **Competitor mention**: Someone mentions a competitor → acknowledge without bashing
10. **Spam/irrelevant**: Someone posts off-topic → redirect politely

Each template should be:
- Under 50 words (social is fast)
- Warm but professional
- Include a natural next step (don't dead-end conversations)
```

## Prompt 12: Social Media Ad Creative Briefs

### Prompt

```
Write creative briefs for 5 social media ad variations promoting [PRODUCT/SERVICE].

Target audience: [TARGET AUDIENCE]
Platform: [PLATFORM]
Budget level: [LOW / MEDIUM / HIGH]
Campaign objective: [AWARENESS / TRAFFIC / CONVERSIONS]

For each variation:
1. **Creative concept**: One sentence describing the visual idea
2. **Primary text**: The ad copy (platform-appropriate length)
3. **Headline**: Bold, benefit-driven
4. **Description**: Supporting line
5. **CTA button**: Most appropriate option
6. **Visual direction**: What the image/video should show (be specific)
7. **Audience angle**: Which segment of your audience this targets

Variation types:
- V1: Problem-aware (they know the pain, don't know you)
- V2: Solution-aware (they know solutions exist, comparing)
- V3: Product-aware (they know you, need a push to buy)
- V4: Retargeting (they visited your page but didn't convert)
- V5: Social proof (lead with results/testimonials)
```

## Prompt 13: Viral Hook Generator

### Prompt

```
Generate 15 viral hooks for social media content about [TOPIC] in the [INDUSTRY] niche.

Target platform: [PLATFORM]
My audience: [TARGET AUDIENCE]

Create hooks in these categories:
1. **Curiosity gap** (3): Hint at something unexpected without revealing it
2. **Bold claim** (3): Make a specific, verifiable statement that challenges convention
3. **"You're doing it wrong"** (3): Call out a common mistake your audience makes
4. **Story opener** (3): Start mid-action to create immediate engagement
5. **Data shock** (3): Lead with a surprising statistic or number

Rules:
- Each hook must be under 15 words
- Must work as the FIRST LINE of a post (before "see more")
- Test readiness: Would this stop YOUR scroll?
- Rate each hook 1-5 on scroll-stop power
```

## Prompt 14: Platform-Specific Repurposing

### Prompt

```
Take this content and repurpose it for 5 platforms:

Original content: [PASTE YOUR BLOG POST, THREAD, VIDEO TRANSCRIPT, OR ARTICLE]

Repurpose into:
1. **Twitter/X thread** (8-10 tweets): Extract key insights, add commentary
2. **LinkedIn post**: Professional angle, personal insight, question ending
3. **Instagram carousel** (8 slides): Visual-friendly, one point per slide, save-worthy
4. **TikTok/Reels script** (30 sec): Hook + 1 key insight + CTA
5. **Newsletter snippet** (150 words): Teaser with link back to full content

Rules:
- Each version must feel NATIVE to the platform (not a copy-paste)
- Adapt the tone: Twitter = sharp/witty, LinkedIn = professional/thoughtful, Instagram = visual/aspirational, TikTok = casual/fast, Newsletter = personal/detailed
- Add platform-specific elements (hashtags, engagement hooks, visual suggestions)
- Each must stand alone — don't assume they've seen the other versions
```

## Pro Tips

- **Batch create**: Use the content calendar prompt on Sunday, schedule everything Monday
- **Hook is everything**: Spend 80% of your time on the first line. The rest doesn't matter if nobody reads past line 1.
- **Engagement begets engagement**: The algorithm rewards posts that get comments in the first hour. Respond to every comment within 60 minutes
- **Test hooks in Stories**: Post your hooks as Stories first. If they get DM replies, turn them into a post
- **Repurpose ruthlessly**: One great idea should become 5+ pieces of content across platforms
