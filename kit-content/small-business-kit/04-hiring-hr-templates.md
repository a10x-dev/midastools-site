## Set Your HR Context

Paste this at the start of every AI session before using HR prompts:

```
You are an HR and recruitment specialist for a small business.

Business: [BUSINESS NAME]
Industry: [INDUSTRY]
Company size: [NUMBER OF EMPLOYEES]
Location: [CITY, STATE]
Remote policy: [FULLY REMOTE / HYBRID / IN-OFFICE]
Company culture: [DESCRIBE IN 2-3 SENTENCES — e.g., "Fast-paced startup. We value ownership, creativity, and direct communication. Casual dress, flexible hours."]
Benefits offered: [LIST KEY BENEFITS — health insurance, PTO, 401k, etc.]
Website: [URL]

Write all HR content in a professional, inclusive, and legally sound manner.
Avoid language that could be discriminatory based on age, gender, race, religion, disability, or any protected class.
```

## Job Description Generator

### Standard Job Description

```
Write a job description for [BUSINESS NAME].

Role: [JOB TITLE]
Department: [DEPARTMENT — if applicable]
Reports to: [MANAGER TITLE]
Location: [CITY, STATE / REMOTE / HYBRID]
Employment type: [FULL-TIME / PART-TIME / CONTRACT]
Salary range: [RANGE OR "Competitive, based on experience"]
Start date: [DATE OR "ASAP"]

Role overview: [2-3 SENTENCES ABOUT WHAT THIS PERSON WILL DO]

Key responsibilities: [LIST 5-8 MAIN TASKS]
- [TASK 1]
- [TASK 2]
- [TASK 3]

Must-have qualifications:
- [REQUIREMENT 1]
- [REQUIREMENT 2]

Nice-to-have qualifications:
- [BONUS 1]
- [BONUS 2]

Requirements:
- Write a compelling job description (300-400 words)
- Start with 2-3 sentences about the company and why this role matters
- Use "you" language ("You will..." not "The candidate will...")
- Include 6-8 responsibilities
- Separate must-have from nice-to-have qualifications
- Include compensation, benefits summary, and location details
- End with how to apply and what to expect in the process
- Use inclusive language — avoid gendered terms, unnecessary jargon, or inflated requirements
- Don't require a degree unless truly necessary for the role
```

### Quick Job Post (Indeed / LinkedIn)

```
Write a concise job listing for [JOB TITLE] at [BUSINESS NAME] optimized for [INDEED / LINKEDIN / OTHER].

Key details:
- Location: [LOCATION]
- Pay: [RANGE]
- Type: [FULL-TIME / PART-TIME]
- Top 3 responsibilities: [LIST THEM]
- Top 3 requirements: [LIST THEM]

Requirements:
- 150-200 words (job board sweet spot)
- Punchy, scannable format with bullet points
- Lead with what makes this role appealing
- Include pay range (listings with pay get 30% more applicants)
- One sentence about company culture
- Clear application instructions
```

### Internal Promotion / Role Change Description

```
Write an internal job posting for [BUSINESS NAME] for a role being offered to existing employees first.

Role: [JOB TITLE]
Current team that will gain this role: [DEPARTMENT]
Why this role is being created: [GROWTH / RESTRUCTURE / NEW NEED]
Who should apply: [CRITERIA FOR INTERNAL CANDIDATES]
Key changes from similar existing roles: [WHAT'S DIFFERENT]

Requirements:
- 150-250 words
- Tone that values internal growth and loyalty
- Clear about new responsibilities and expectations
- Mention any changes in compensation or title
- Include the internal application process and deadline
- Encourage candidates to talk to [MANAGER NAME] with questions
```

## Interview Question Generator

### General Interview Questions (Any Role)

```
Generate interview questions for a [JOB TITLE] position at [BUSINESS NAME].

Create 3 questions for each category:

1. **Culture fit** — Do they align with our values of [LIST 2-3 VALUES]?
2. **Problem-solving** — Can they think on their feet?
3. **Role-specific skills** — Can they actually do [KEY SKILL 1] and [KEY SKILL 2]?
4. **Work style** — How do they handle [RELEVANT SCENARIO — deadlines, teamwork, ambiguity, remote work]?
5. **Growth mindset** — Are they eager to learn and improve?

For each question:
- Write the question
- Explain what a strong answer looks like (2-3 sentences)
- Note any red flags to watch for
```

### Behavioral Interview Questions (STAR Method)

```
Generate 10 behavioral interview questions for a [JOB TITLE] at [BUSINESS NAME] using the STAR method (Situation, Task, Action, Result).

Focus areas:
- [COMPETENCY 1 — e.g., leadership, customer service, initiative]
- [COMPETENCY 2 — e.g., conflict resolution, time management, creativity]
- [COMPETENCY 3 — e.g., adaptability, teamwork, attention to detail]

For each question:
- Write the behavioral question starting with "Tell me about a time when..."
- Provide a scoring rubric (what does a 1, 3, and 5 look like?)
- Include a follow-up probe question
```

### Technical / Skills Assessment

```
Design a practical skills assessment for a [JOB TITLE] candidate at [BUSINESS NAME].

Key skills to test:
1. [SKILL 1]
2. [SKILL 2]
3. [SKILL 3]

Requirements:
- Create a 30-60 minute take-home or in-person exercise
- Make it relevant to actual work they'd do in this role
- Include clear instructions the candidate receives
- Include a scoring rubric for evaluators
- Should be fair to candidates from different backgrounds
- Don't require proprietary tools or paid software
- Include a note on reasonable accommodations
```

### Interview Scorecard

```
Create an interview scorecard for evaluating [JOB TITLE] candidates at [BUSINESS NAME].

Evaluation criteria:
1. [CRITERION 1 — e.g., technical skills]
2. [CRITERION 2 — e.g., communication]
3. [CRITERION 3 — e.g., culture fit]
4. [CRITERION 4 — e.g., problem-solving]
5. [CRITERION 5 — e.g., relevant experience]

For each criterion:
- Define what a 1 (poor), 3 (meets expectations), and 5 (exceptional) looks like
- Include space for notes

Also include:
- Overall recommendation: Strong Hire / Hire / No Hire / Strong No Hire
- Concerns or flags section
- "Would I want to work with this person daily?" yes/no
```

## Offer Letter Templates

### Standard Offer Letter

```
Write a formal job offer letter from [BUSINESS NAME].

Candidate: [CANDIDATE NAME]
Position: [JOB TITLE]
Department: [DEPARTMENT]
Start date: [DATE]
Salary: [AMOUNT — annual or hourly]
Pay schedule: [WEEKLY / BI-WEEKLY / MONTHLY]
Employment type: [FULL-TIME / PART-TIME / CONTRACT]
Location: [OFFICE ADDRESS / REMOTE]
Reports to: [MANAGER NAME AND TITLE]
Benefits: [LIST — health, dental, PTO days, 401k, etc.]
Equity or bonus: [DETAILS IF APPLICABLE]
Contingencies: [BACKGROUND CHECK / DRUG TEST / REFERENCE CHECK / NONE]
Response deadline: [DATE]

Requirements:
- Professional, warm, and exciting tone
- Open with congratulations and enthusiasm
- Clearly state all compensation and benefits
- Include start date, location, and reporting structure
- List any contingencies
- Include at-will employment language (standard in most US states)
- Request signature and return by deadline
- Provide contact for questions
- 300-400 words
- Add a note: "This template is a starting point. Have an employment attorney review your offer letters for compliance with local laws."
```

### Informal Offer (Pre-Letter)

```
Write a brief, exciting email from [YOUR NAME] at [BUSINESS NAME] to [CANDIDATE NAME] letting them know they got the job.

Purpose: This is the informal "good news" email before the formal letter follows.

Details:
- Position: [JOB TITLE]
- Key detail to mention: [ONE EXCITING THING — salary, start date, team they'll join]

Requirements:
- 60-100 words
- Enthusiastic but professional
- Tell them the formal offer letter is coming within [TIMEFRAME]
- Mention who to contact with questions
- Express genuine excitement about them joining
```

## Rejection Email Templates

### After Resume Screen (No Interview)

```
Write a rejection email from [BUSINESS NAME] for a candidate who was not selected for an interview.

Candidate: [NAME]
Position they applied for: [JOB TITLE]
Number of applicants for this role: [APPROXIMATE NUMBER, if you want to include this]

Requirements:
- Kind, respectful, and concise
- Thank them for their time and interest
- Briefly acknowledge something positive if possible
- Clearly state they won't be moving forward for this role
- Encourage them to apply for future openings if appropriate
- Do NOT give specific reasons (legal risk)
- 60-80 words
- Genuine, not robotic
```

### After Interview

```
Write a rejection email from [BUSINESS NAME] for a candidate who interviewed but was not selected.

Candidate: [NAME]
Position: [JOB TITLE]
Interview stage reached: [PHONE SCREEN / FIRST ROUND / FINAL ROUND]
Something specific they did well: [POSITIVE NOTE — e.g., "Your presentation on X was impressive"]
Include feedback: [YES — provide constructive feedback / NO — keep it general]

If yes, feedback topic: [WHAT TO MENTION — e.g., "they were strong technically but lacked the management experience we need"]

Requirements:
- Warm, personal, and respectful
- Thank them for investing their time in the process
- Mention something specific and positive about their candidacy
- Clearly state the decision
- If providing feedback: be constructive, specific, and kind
- If not providing feedback: keep it general ("we went with a candidate whose experience more closely matched")
- Encourage them to stay connected
- Offer to keep them in mind for future roles
- 80-120 words
```

### After Final Round (Very Close Decision)

```
Write a rejection email from [BUSINESS NAME] for a finalist who was the second choice.

Candidate: [NAME]
Position: [JOB TITLE]
What made them a strong candidate: [SPECIFIC STRENGTHS]

Requirements:
- This person was almost hired — the tone should reflect how close they came
- Express genuine admiration and respect for their candidacy
- Be honest that it was a very close decision
- Offer to keep them at the top of your list for future opportunities
- If appropriate, offer to connect on LinkedIn or provide a referral
- 100-150 words
- Feels personal, not templated
```

## Employee Handbook Section Generator

### Generate Any Handbook Section

```
Write an employee handbook section for [BUSINESS NAME] on the topic of [SECTION TOPIC].

Topics (choose one):
- Welcome and company overview
- Equal opportunity employment
- At-will employment
- Code of conduct and workplace behavior
- Anti-harassment and anti-discrimination policy
- Working hours and attendance
- Remote work policy
- Paid time off (PTO) and holidays
- Leave of absence (FMLA, parental, bereavement)
- Compensation and pay schedule
- Benefits overview
- Performance reviews and promotions
- Disciplinary process
- Social media policy
- Confidentiality and intellectual property
- Health and safety
- Termination and offboarding
- Acknowledgment page

Company details:
- Size: [NUMBER OF EMPLOYEES]
- State: [STATE — laws vary by state]
- Industry: [INDUSTRY]
- Remote/hybrid/in-office: [WORK MODEL]

Requirements:
- Professional, clear language anyone can understand
- Include relevant legal disclaimers where appropriate
- Note where state-specific or industry-specific laws may apply
- 200-350 words per section
- Include a disclaimer: "This is a template and should be reviewed by an employment attorney familiar with [STATE] law."
```

### Complete Mini-Handbook (Quick Version)

```
Write a concise employee handbook for [BUSINESS NAME], a small business with [NUMBER] employees in [STATE].

Include these essential sections:
1. Welcome and company mission (50 words)
2. At-will employment statement (30 words)
3. Equal opportunity and anti-harassment policy (100 words)
4. Working hours and attendance expectations (80 words)
5. PTO policy: [NUMBER] days per year, [POLICY DETAILS] (80 words)
6. Code of conduct (100 words)
7. Social media and confidentiality (80 words)
8. Disciplinary process (80 words)
9. Termination procedures (60 words)
10. Acknowledgment signature page

Total target: 800-1,000 words
Include a cover page with company name, effective date, and version number.
Add a disclaimer about legal review before implementation.
```

## Performance Review Template

### Annual / Semi-Annual Performance Review

```
Create a performance review template for [BUSINESS NAME] to evaluate a [JOB TITLE].

Review period: [START DATE] to [END DATE]
Employee name: [NAME — or leave blank for template]
Reviewer: [MANAGER NAME/TITLE]

Include these sections:

1. **Overall rating** — Scale of 1-5 with clear definitions:
   - 1: Below expectations
   - 2: Partially meets expectations
   - 3: Meets expectations
   - 4: Exceeds expectations
   - 5: Exceptional

2. **Key responsibilities review** — Rate performance on each core responsibility
   Responsibilities for this role:
   - [RESPONSIBILITY 1]
   - [RESPONSIBILITY 2]
   - [RESPONSIBILITY 3]

3. **Goals from last review** — Status of previous goals (Met / In Progress / Not Met)

4. **Strengths** — Space for 2-3 specific strengths with examples

5. **Areas for improvement** — Space for 1-2 growth areas with specific, constructive guidance

6. **Goals for next period** — 3-5 SMART goals

7. **Employee self-assessment** — Space for the employee's perspective

8. **Development plan** — Training, skills, or experiences needed for growth

9. **Compensation discussion** — Notes on raise, bonus, or promotion (if applicable)

10. **Signatures** — Employee and manager with date

Requirements:
- Professional and constructive tone throughout
- Include sample language for each section showing what a good review looks like
- Balanced — don't avoid difficult feedback but frame it constructively
- Focus on observable behaviors and outcomes, not personality traits
```

### Quick Check-In Template (Monthly / Quarterly)

```
Create a brief performance check-in template for [BUSINESS NAME] managers to use monthly or quarterly.

Purpose: Lightweight alternative to formal reviews that keeps communication flowing.

Include:
1. What went well this period? (manager notes + employee notes)
2. What could have gone better? (manager notes + employee notes)
3. Progress on current goals (brief status update)
4. Roadblocks or challenges (and how the manager can help)
5. Priorities for next period (agree on top 3)
6. Career development: "What skill do you want to build next?"
7. One-line pulse check: "On a scale of 1-10, how are you feeling about your work right now?"

Requirements:
- Fits on one page
- Takes 15-30 minutes to complete
- Conversational, not bureaucratic
- Equal space for manager and employee input
```

### Writing Performance Feedback

```
Help me write performance feedback for an employee at [BUSINESS NAME].

Employee: [NAME]
Role: [JOB TITLE]
What they did well: [LIST ACCOMPLISHMENTS OR STRENGTHS]
Where they need to improve: [LIST AREAS]
Specific example of concern: [DESCRIBE A SPECIFIC SITUATION]
Overall assessment: [BELOW / MEETS / EXCEEDS EXPECTATIONS]

Requirements:
- Write the feedback in a professional, constructive tone
- Use the SBI model: Situation, Behavior, Impact
- For strengths: be specific with examples, not vague ("great team player")
- For improvement areas: describe the behavior (not the person), the impact, and what "good" looks like
- Include 2-3 suggested goals for improvement
- Balance honesty with kindness — this person should leave the review motivated, not crushed
- 200-300 words
```
