Store: [STORE NAME]
Niche: [YOUR NICHE]

Price range: [LOW - HIGH]

## Facebook / Instagram Ads

### Prompt 1: Single Image Ad — Problem/Solution

```
Write a Facebook/Instagram single image ad for [PRODUCT NAME] from [STORE NAME].

Product details:
- What it does: [PRIMARY FUNCTION]
- Problem it solves: [CUSTOMER PAIN POINT]
- Key benefit: [MAIN BENEFIT]
- Price: [PRICE]
- Target audience: [WHO SEES THIS AD]
- Landing page: [URL]

Write 3 versions using the Problem-Agitate-Solution framework:
For each version provide:
- Primary text (above the image): 125 characters max for the hook line, then 2-3 additional sentences (total 40-80 words)
- Headline (below the image): 40 characters max
- Description: 30 characters max
- CTA button: [SHOP NOW / LEARN MORE / GET OFFER]

Version 1: Emotional angle — focus on how the problem feels
Version 2: Logical angle — focus on facts and data
Version 3: Social proof angle — focus on what others experience

Suggest an image concept for each version.
```

> **Why this works:** The Problem-Agitate-Solution framework is the highest-converting ad structure because it mirrors the buyer's thought process: "I have this problem" > "It's even worse than I thought" > "Here's the solution." Matching their internal dialogue creates instant relevance.

### Prompt 3: Video Ad Script — Product Demo

```
Write a Facebook/Instagram video ad script for [PRODUCT NAME] from [STORE NAME].

Product details:
- What it does: [FUNCTION]
- Key benefit: [MAIN BENEFIT]
- Visual appeal: [WHAT LOOKS IMPRESSIVE ABOUT THE PRODUCT]
- Price: [PRICE]
- Target audience: [WHO SEES THIS AD]

Video length: [15 SECONDS / 30 SECONDS / 60 SECONDS]

Script format:
- HOOK (first 3 seconds): The scroll-stopping opening — what the viewer sees and hears
- PROBLEM (seconds 3-8): Show/describe the pain point
- SOLUTION (seconds 8-18): Introduce the product in action
- PROOF (seconds 18-25): Social proof, results, or testimonial
- CTA (final 5 seconds): Clear next step

For each section provide:
- On-screen text overlay
- Voiceover script (or text-to-speech script)
- Visual direction (what's being shown)
- Music/sound mood suggestion

Write 2 hook variations to test.
```

> **Pro tip:** Facebook's data shows that 65% of people who watch the first 3 seconds will watch for at least 10 more. Your hook is everything. Test multiple hooks with the same body — it's the highest-leverage A/B test in video advertising.

## Google Shopping Titles & Descriptions

### Prompt 6: Google Shopping Product Title Optimization

```
Write optimized Google Shopping titles for these products from [STORE NAME].

Products:
1. [CURRENT TITLE 1] — [BRAND] — [KEY ATTRIBUTES: color, size, material]
2. [CURRENT TITLE 2] — [BRAND] — [KEY ATTRIBUTES]
3. [CURRENT TITLE 3] — [BRAND] — [KEY ATTRIBUTES]
4. [CURRENT TITLE 4] — [BRAND] — [KEY ATTRIBUTES]
5. [CURRENT TITLE 5] — [BRAND] — [KEY ATTRIBUTES]

Requirements for each title:
- Maximum 150 characters (aim for 70-100 for best display)
- Structure: [Brand] + [Product Type] + [Key Attributes] + [Size/Color if relevant]
- Front-load the most searched terms
- Include material, color, and size when applicable
- Remove subjective words (best, amazing, premium) — Google penalizes them
- Include the target keyword naturally
- Write 2 versions: one for Google Shopping, one for Google Search ads

Target keywords per product:
1. [TARGET KEYWORD 1]
2. [TARGET KEYWORD 2]
3. [TARGET KEYWORD 3]
4. [TARGET KEYWORD 4]
5. [TARGET KEYWORD 5]
```

> **Why this works:** Google Shopping titles are the most important factor in whether your product appears for a search. Google matches search queries to your title. A title that says "Blue Cotton V-Neck T-Shirt Men's Size L" will show up for far more searches than "The Classic Tee."

### Prompt 7: Google Shopping Product Description

```
Write Google Shopping product descriptions for [STORE NAME].

Products:
1. [PRODUCT 1 NAME]: [KEY FEATURES], [MATERIAL], [SIZE/DIMENSIONS], [TARGET USE]
2. [PRODUCT 2 NAME]: [KEY FEATURES], [MATERIAL], [SIZE/DIMENSIONS], [TARGET USE]
3. [PRODUCT 3 NAME]: [KEY FEATURES], [MATERIAL], [SIZE/DIMENSIONS], [TARGET USE]

Requirements for each description:
- 500-1000 characters (Google's limit is 5000 but shorter performs better)
- First 175 characters are the most important (visible in product listing)
- Include primary and secondary keywords naturally
- Lead with the most important product information
- Include material, dimensions, compatibility, and use cases
- No promotional language (no "sale," "free shipping," "best price")
- No ALL CAPS
- No links or references to your store
- Factual, descriptive, and keyword-rich

Primary keywords:
1. [KEYWORD 1]
2. [KEYWORD 2]
3. [KEYWORD 3]
```

## TikTok Ad Scripts

### Prompt 9: TikTok In-Feed Ad Script — Hook-Based

```
Write a TikTok in-feed ad script for [PRODUCT NAME] from [STORE NAME].

Product details:
- What it is: [PRODUCT]
- Key benefit: [MAIN BENEFIT]
- Price: [PRICE]
- Target audience on TikTok: [AGE, INTERESTS, BEHAVIOR]
- Visual style: [UGC / POLISHED / TUTORIAL / COMEDY / SATISFYING]

Duration: [15 / 30 / 60] seconds

Write 3 scripts with different hooks:

Script 1 — "I can't believe" hook:
- Hook: Start with disbelief or surprise about the product
- Body: Quick demo or reveal
- CTA: Where to buy

Script 2 — "POV" hook:
- Hook: "POV: you finally found [SOLUTION TO PROBLEM]"
- Body: Show the transformation or experience
- CTA: Link in bio

Script 3 — "Things TikTok made me buy" hook:
- Hook: Trend-aware, community-driven opening
- Body: Honest review format
- CTA: "Link in my bio if you need this"

For each script provide:
- On-screen text (word-by-word timing)
- Voiceover or dialogue
- Visual direction
- Trending sound suggestion or music mood
- Hashtag suggestions (5-7)
```

> **Pro tip:** TikTok's algorithm favors ads that look like organic content. The more your ad feels like a regular TikTok — shaky camera, natural lighting, real person talking — the better it performs. Polished studio ads get skipped.

## Retargeting Ad Copy

### Prompt 16: Website Visitor Retargeting — Browsed But Didn't Buy

```
Write retargeting ad copy for [STORE NAME] targeting people who visited the website but didn't purchase.

Context:
- They browsed your store (general visitors, not specific product viewers)
- They left without adding to cart
- Time since visit: [1-3 DAYS / 3-7 DAYS / 7-14 DAYS]
- Platform: [FACEBOOK / INSTAGRAM / GOOGLE DISPLAY]

Store differentiators:
- [USP 1 — e.g., free shipping over $50]
- [USP 2 — e.g., 30-day returns]
- [USP 3 — e.g., 10,000+ 5-star reviews]

Requirements:
Write 3 ad variations:

Variation 1 — Bestseller showcase:
- "Our customers' favorites" angle
- Feature [3 BESTSELLING PRODUCTS]
- Primary text: 40-60 words
- Headline + description

Variation 2 — Social proof:
- Lead with reviews, ratings, or customer count
- "Join [X] happy customers" framing
- Primary text: 40-60 words
- Headline + description

Variation 3 — Brand story reminder:
- Remind them why you're different
- Share your mission or value proposition
- Primary text: 40-60 words
- Headline + description

No discount in these ads — save discounts for deeper-funnel retargeting.
```

> **Pro tip:** Segment your retargeting by recency. People who visited 1-3 days ago need a gentle reminder. People who visited 7-14 days ago need a stronger hook. People who visited 14-30 days ago may need an incentive. Matching the message to the time gap dramatically improves ROAS.

### Prompt 19: Flash Sale Ad — Multi-Platform

```
Write flash sale ad copy for [STORE NAME] across multiple platforms.

Sale details:
- Discount: [PERCENTAGE / DOLLAR OFF / BOGO / FREE SHIPPING]
- Duration: [HOURS / DAYS]
- Start: [DATE + TIME]
- End: [DATE + TIME]
- Code (if needed): [CODE]
- Products included: [ALL / SPECIFIC CATEGORIES / SPECIFIC PRODUCTS]
- Exclusions: [WHAT'S NOT INCLUDED]

Write ads for each platform:

Facebook/Instagram (3 versions):
- Version 1: Urgency-driven (50-70 words primary text)
- Version 2: Value-driven (50-70 words primary text)
- Version 3: FOMO-driven (50-70 words primary text)
- Headlines and descriptions for each

Google Search (1 responsive search ad):
- 10 headlines (30 chars each)
- 4 descriptions (90 chars each)

TikTok (1 script):
- 15-second script
- Hook + offer + CTA
- On-screen text callouts

Pinterest (1 pin):
- Pin title + description
- Gift/self-purchase angle

All ads must clearly state: the offer, the deadline, and how to redeem.
```

> **Why this works:** Flash sales create urgency that drives immediate action. But the urgency must be real — if you run "flash sales" every week, customers learn to wait. Use flash sales 4-6 times per year for maximum impact. Multi-platform amplifies reach without additional creative effort.

### Prompt 25: Ad Copy Refresh — Revive Fatigued Ads

```
Rewrite this fatigued ad for [STORE NAME] to give it new life.

Current ad that's declining:
- Platform: [PLATFORM]
- Current primary text: [PASTE CURRENT COPY]
- Current headline: [HEADLINE]
- Current performance: CTR dropped from [X]% to [X]% over [TIMEFRAME]
- Target audience: [AUDIENCE]
- Product: [PRODUCT]

Requirements:
Write 3 fresh variations that:
- Keep the same product and offer
- Change the angle completely (different hook, different emotional trigger)
- Maintain brand voice consistency

Variation 1 — New emotional angle:
- If the original was logical, go emotional (or vice versa)
- New primary text + headline + description

Variation 2 — New format:
- If the original was long-form, try ultra-short (or vice versa)
- New primary text + headline + description

Variation 3 — New social proof:
- Lead with a customer quote, stat, or community number
- New primary text + headline + description

Explain the strategic reasoning behind each variation.
```

> **Why this works:** Ad fatigue is the #1 killer of profitable campaigns. When CTR drops, most advertisers increase budget or change targeting. Instead, refresh the creative. Your audience hasn't changed — they've just seen your current ad too many times. New angles to the same audience costs less than new audiences for old ads.
