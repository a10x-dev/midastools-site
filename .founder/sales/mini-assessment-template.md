## DELIVERY MECHANICS

### Format
- **PDF only.** Do not send raw markdown or Google Doc links — feels less polished.
- **Notion → "Export as PDF"** is fastest (15 min from filled template to file).
- **Filename:** `mini-assessment-[firstname]-[YYYYMMDD].pdf` (e.g., `mini-assessment-hiedeh-20260502.pdf`)

### Send via Resend
Use the existing `.founder/tools/send-pitches.py` pattern. Edit a one-off send:

**NOTE:** `send-pitches.py` does not currently support attachments — needs a 1-line addition before first delivery. Resend supports attachments natively via base64.

CAPABILITY_GAP: send-pitches.py does not support attachments | Blocks 4-hr SLA on first reply-type-B fire — must extend tool before May 2 audit window OR fall back to manual Resend dashboard send

1. Open Resend dashboard manually (resend.com)
2. Compose email with attached PDF
3. Send to single recipient
4. Log in audit-replies-tracker.md

Manual fallback adds ~10 min vs scripted send. Acceptable for first 2-3 deliveries.
