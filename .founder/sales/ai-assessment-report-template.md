## QA CHECKLIST (for analyst before delivery)

- [ ] Every recommendation references something the client actually said
- [ ] At least 1 &ldquo;connect-the-dots&rdquo; insight per the framework
- [ ] At least 1 recommendation feels &ldquo;obvious in hindsight&rdquo;
- [ ] Rejection list is 3-5 tools with specific-to-them reasons
- [ ] No tool we haven&apos;t personally tested in similar context
- [ ] Implementation order is unambiguous
- [ ] Year-1 ROI projection on a single line, conservative assumptions
- [ ] Upsell ladder visible but soft (no urgency, no scarcity)
- [ ] PDF renders cleanly on mobile + desktop
- [ ] Cover image personalized
- [ ] Page count between 15-25 (not less, not more)
- [ ] Spell-checked, brand-checked, link-checked
